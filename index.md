With the development of the industry, there are many low voltage and high current application scenarios, such as data center and  CPU and so on.  The power supply requirements of these applications pose great challenges to power electronics.  Multi-phase interleave parallel technology is the key factor to solve this problem, but for multi-phase system, the temperature mismatch between modules will seriously affect the system life.  Therefore, thermal equalization control strategy is needed to improve the operation of multi-phase interleaved parallel system, so as to prolong the system life and improve the system reliability.
